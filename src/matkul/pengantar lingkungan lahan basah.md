# Kesimpulan
**LibreOffice Writer** pada sistem operasi Ubuntu berfungsi dengan baik untuk membuat makalah akademik. Tidak ditemukan kendala selama proses pembuatan, penyimpanan, maupun ekspor file ke format lain.
